### SYSTEMIC FIX — AC header parser (2026-03-16T04:41:00Z)

- **Root cause found**: `validateProofQuality()` in `src/lib/verify.ts` used regex `/^## AC \d+:/gm` (with space), but showboat proof documents use `## AC1:` (no space). This caused 0/0 ACs for ALL proof documents.
- **Fix applied**: Changed regex to `/^## AC ?(\d+):/gm` to accept both formats.
- **Test added**: New test case in `src/lib/__tests__/verify.test.ts` for no-space format.
- **Impact**: All 1438 tests pass. This fix unblocks verification for every verified story in the sprint.
